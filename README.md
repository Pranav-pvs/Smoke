# 💨 CSS Smoke Animation

A lightweight and visually appealing **Smoke Animation** created using **HTML5** and **CSS3**. This project demonstrates how CSS keyframe animations, transforms, opacity, and scaling can be combined to simulate realistic smoke without using JavaScript.

## ✨ Features

* 💨 Pure HTML & CSS smoke animation
* 🎬 Smooth infinite animation loop
* 🎨 Realistic fading and scaling effects
* 🚀 No JavaScript required
* 📱 Lightweight and responsive
* 🎯 Beginner-friendly CSS animation project

## 🛠️ Technologies Used

* HTML5
* CSS3

  * CSS Animations
  * Keyframes
  * Transforms

  * Absolute Positioning
